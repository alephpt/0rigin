## IMPLEMENTATION GUIDANCE

### For Each Chapter, Ask:

1. **Is this a physics claim?**
   - If yes: Brief, honest grounding statement
   - If no: Let philosophical speculation stand

2. **Does "we discovered" mean computational validation?**
   - If yes: Keep it
   - If no: Change to "we propose"

3. **Is this breaking narrative flow with technical detail?**
   - If yes: Reference TRD paper instead
   - If no: Brief mention is fine

4. **Is this philosophical depth or physics overclaim?**
   - Depth: Preserve it
   - Overclaim: Add honesty marker
